\moduledescription{mod.selfprotect.memory.desc}

The module \tds{mod.selfprotect.memory} of subsystem \tdslink{sub.selfprotect}
provides mechanisms for sanitizing system memory. 

\moduleprocesses{prc.selfprotect.memory}

\moduleprocess{prc.selfprotect.memory.wipemem}{Overwriting Sensitive Data}

Sensitive data is overwritten by pseudo-random numbers generated by the TOE's
random number generator in \tdslink[fq]{mod.cryptsystem.rng}. The RNG is called
via \tdslink[fq]{int.cryptsystem.rng.creaternd}.

\implementedsfr{\sfrlink{fdp_rip.1}}

\moduleinterfaces{int.selfprotect.memory}

\moduleinterfaceprovided{int.selfprotect.memory.cleanup}

This interface triggers the memory sanitization (see \sectref{prc.selfprotect.memory.wipemem}).
